# 题后总结

## 知识点
原地标记 + 利用首行首列作为标记位

## 思路
1. **核心思想**：用矩阵的第一行和第一列作为标记数组，记录哪些行列需要置零
2. **第一步**：遍历整个矩阵，遇到 0 时：
   - 记录首行/首列是否需要置零（用两个布尔变量）
   - 在对应的首行和首列位置标记 0
3. **第二步**：根据首行首列的标记，将对应行列置零（从索引 1 开始，避开首行首列）
4. **第三步**：处理首行和首列本身

## 核心逻辑
```typescript
// 第一步：标记
for (let m = 0; m < matrix.length; m++) {
    for (let n = 0; n < matrix[m].length; n++) {
        if (matrix[m][n] === 0) {
            if (m === 0) isFirstRowHasZero = true;
            if (n === 0) isFirstColHasZero = true;
            matrix[0][n] = 0;  // 标记列
            matrix[m][0] = 0;  // 标记行
        }
    }
}

// 第二步：根据标记置零（避开首行首列）
for (let n = 1; n < matrix[0].length; n++) {
    if (matrix[0][n] === 0) {
        for (let m = 1; m < matrix.length; m++) {
            matrix[m][n] = 0;
        }
    }
}

for (let m = 1; m < matrix.length; m++) {
    if (matrix[m][0] === 0) {
        for (let n = 1; n < matrix[m].length; n++) {
            matrix[m][n] = 0;
        }
    }
}

// 第三步：处理首行首列
if (isFirstRowHasZero) { ... }
if (isFirstColHasZero) { ... }
```

## 关键点
1. **两个布尔变量**：`isFirstRowHasZero` 和 `isFirstColHasZero` 记录首行首列是否需要置零
2. **标记顺序**：先记录首行首列状态，再设置标记位，避免覆盖
3. **处理顺序**：先处理内部行列（从索引 1 开始），最后处理首行首列
4. **空间优化**：只用 O(1) 额外空间（两个布尔变量）

## 易错点
1. **首行首列的处理时机**：必须最后处理，否则标记会被提前清零
2. **遍历范围**：第二步遍历行列时从索引 1 开始，避开首行首列
3. **标记冲突**：如果先清零首行首列，会丢失标记信息
4. **边界条件**：单行或单列矩阵需要正确处理

## 复杂度
- 时间复杂度：O(m × n)，遍历矩阵常数次
- 空间复杂度：O(1)，只使用了两个布尔变量

## 示例演示
以 `matrix = [[1,1,1],[1,0,1],[1,1,1]]` 为例：

**第一步：标记**

遍历发现 `matrix[1][1] = 0`：
- `m !== 0` 且 `n !== 0`，不更新布尔变量
- 设置 `matrix[0][1] = 0`（标记第 1 列）
- 设置 `matrix[1][0] = 0`（标记第 1 行）

标记后矩阵：
```
[[1, 0, 1],
 [0, 0, 1],
 [1, 1, 1]]
```

**第二步：根据标记置零**

- 检查首行：`matrix[0][1] = 0` → 第 1 列全部置零
- 检查首列：`matrix[1][0] = 0` → 第 1 行全部置零

结果：
```
[[1, 0, 1],
 [0, 0, 0],
 [1, 0, 1]]
```

**第三步：处理首行首列**

- `isFirstRowHasZero = false`，首行不变
- `isFirstColHasZero = false`，首列不变

最终结果：
```
[[1, 0, 1],
 [0, 0, 0],
 [1, 0, 1]] ✓
```

## 算法对比

| 方法 | 时间复杂度 | 空间复杂度 | 特点 |
|------|-----------|-----------|------|
| 额外集合 | O(m×n) | O(m+n) | 简单直观 |
| 首行首列标记 | O(m×n) | **O(1)** | 最优解，面试常考 |
