# 题后总结

## 知识点
滑动窗口 + 哈希表（字符频次统计）

## 思路
1. 统计字符串 p 中每个字符的频次
2. 使用固定大小为 p.length 的滑动窗口在 s 上移动
3. 维护窗口内字符的频次，与 p 的频次对比
4. 当窗口内字符频次与 p 完全相同时，记录起始位置

## 关键点
1. 不需要每次都完整比较两个频次表，可以维护一个变量记录匹配的字符数
2. 窗口大小固定为 p.length，每次向右移动一位
3. 进入窗口的字符频次 +1，移出窗口的字符频次 -1
4. 时间复杂度 O(n)，空间复杂度 O(1)（字符集大小固定）

## 易错点
1. s 长度小于 p 时直接返回空数组
2. p 为空字符串时的边界处理
3. 频次表的更新顺序：先添加新字符，再移除旧字符
4. 比较频次时要检查所有字符，不能只检查部分

## 优化方向
可以使用单个频次表 + 计数器来优化，避免每次都比较整个频次表
