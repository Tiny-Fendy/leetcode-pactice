# 题后总结

## 知识点
数组操作 + 三次反转法（最优解）

## 思路（三次反转法）
1. **边界处理**：k 大于数组长度时取模 `k = k % nums.length`；k 为 0 时直接返回
2. **第一次反转**：反转整个数组 `[0, n-1]`
3. **第二次反转**：反转前 k 个元素 `[0, k-1]`
4. **第三次反转**：反转后 n-k 个元素 `[k, n-1]`
5. **原理**：通过三次局部反转实现整体轮转，无需额外空间

## 核心逻辑
```typescript
const reverse = (start: number, end: number) => {
    while (start < end) {
        [nums[start], nums[end]] = [nums[end], nums[start]];
        start++;
        end--;
    }
};

reverse(0, nums.length - 1);  // 反转整个数组
reverse(0, k - 1);             // 反转前 k 个
reverse(k, nums.length - 1);   // 反转后 n-k 个
```

## 关键点
1. **反转函数**：使用双指针从两端向中间交换元素
2. **三次反转的顺序**：必须先反转整体，再分别反转前后两部分
3. **原地修改**：直接在原数组上操作，不需要额外空间
4. **k 取模**：处理 k 大于数组长度的情况

## 易错点
1. **k 为 0 的边界**：需要单独处理，避免不必要的计算
2. **反转范围**：注意区间的开闭，`[start, end]` 是闭区间
3. **交换顺序**：双指针相向移动，直到相遇为止
4. **解构赋值**：使用 `[a, b] = [b, a]` 简洁地交换两个元素

## 复杂度
- 时间复杂度：O(n)，每个元素最多被访问 2 次
- 空间复杂度：O(1)，只使用了常数额外空间

## 示例演示
以 `nums = [1,2,3,4,5,6,7], k = 3` 为例：

**步骤 1**：反转整个数组 `[0, 6]`
```
[1,2,3,4,5,6,7] → [7,6,5,4,3,2,1]
```

**步骤 2**：反转前 k=3 个元素 `[0, 2]`
```
[7,6,5,4,3,2,1] → [5,6,7,4,3,2,1]
```

**步骤 3**：反转后 n-k=4 个元素 `[3, 6]`
```
[5,6,7,4,3,2,1] → [5,6,7,1,2,3,4] ✓
```

最终结果：`[5,6,7,1,2,3,4]`

## 算法对比

### 方法一：切片+移位（旧解法）
```typescript
const tem = nums.slice(-k);
for (let i = nums.length - 1; i >= 0; i--) {
    if (i > k - 1) {
        nums[i] = nums[i - k];
    } else {
        nums[i] = tem[i];
    }
}
```
- 时间复杂度：O(n)
- 空间复杂度：O(k)
- 优点：直观易懂
- 缺点：需要额外空间存储尾部元素

### 方法二：三次反转（新解法，推荐）
```typescript
reverse(0, nums.length - 1);
reverse(0, k - 1);
reverse(k, nums.length - 1);
```
- 时间复杂度：O(n)
- 空间复杂度：O(1) ✓
- 优点：最优空间复杂度，代码简洁
- 缺点：需要理解反转的原理

| 方法 | 时间复杂度 | 空间复杂度 | 代码复杂度 | 推荐度 |
|------|-----------|-----------|-----------|--------|
| 三次反转 | O(n) | **O(1)** | 简单 | ⭐⭐⭐⭐⭐ |
| 切片+移位 | O(n) | O(k) | 中等 | ⭐⭐⭐ |
| 环形替换 | O(n) | O(1) | 复杂 | ⭐⭐⭐⭐ |
| 额外数组 | O(n) | O(n) | 简单 | ⭐⭐ |

## 为什么三次反转有效？

数学原理：
- 原数组：`[A, B]`，其中 A 是前 n-k 个元素，B 是后 k 个元素
- 目标：`[B, A]`
- 步骤：
  1. 整体反转：`[A, B]` → `reverse([A, B])` = `[reverse(B), reverse(A)]`
  2. 反转前 k 个：`reverse(reverse(B))` = `B`
  3. 反转后 n-k 个：`reverse(reverse(A))` = `A`
  4. 结果：`[B, A]` ✓
