# 题后总结

## 知识点
动态规划（线性 DP）

## 思路
1. **问题分析**：不能偷相邻的房屋，求最大收益
2. **状态定义**：`dp[i]` 表示考虑前 `i+1` 个房屋时能偷到的最大金额
3. **状态转移**：对于每个房屋，有两种选择：
   - 不偷当前房屋：最大金额 = `dp[i-1]`
   - 偷当前房屋：最大金额 = `dp[i-2] + nums[i]`
   - 取两者最大值：`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`
4. **空间优化**：只需用两个变量维护前两个状态

## 关键点
1. **边界条件**：
   - 空数组：返回 0
   - 单个房屋：返回 `nums[0]`
   - 两个房屋：返回 `max(nums[0], nums[1])`
2. **状态压缩**：用 `prev2` 和 `prev1` 代替整个 dp 数组
3. **滚动更新**：每次迭代后，`prev2 = prev1; prev1 = curr`

## 核心公式
```
dp[i] = max(dp[i-1], dp[i-2] + nums[i])
```

## 易错点
1. 空数组需要特殊处理，避免访问 `nums[0]` 报错
2. 状态更新顺序：先计算当前值，再滚动更新前两个状态
3. 理解"不相邻"的约束：偷了第 `i` 个，就不能偷第 `i-1` 个，但可以偷第 `i-2` 个

## 复杂度
- 时间复杂度：O(n)，只需一次遍历
- 空间复杂度：O(1)，只使用了常数个变量

## 示例演示
以 `nums = [2, 7, 9, 3, 1]` 为例：

| i | nums[i] | prev2 | prev1 | curr = max(prev1, prev2+nums[i]) |
|---|---------|-------|-------|----------------------------------|
| 0 | 2       | 0     | 0     | 2                                |
| 1 | 7       | 0     | 2     | 7                                |
| 2 | 9       | 2     | 7     | 11                               |
| 3 | 3       | 7     | 11    | 11                               |
| 4 | 1       | 11    | 11    | 12 ✓                             |

最终结果：12（偷第 1、3、5 号房屋：2 + 9 + 1 = 12）
