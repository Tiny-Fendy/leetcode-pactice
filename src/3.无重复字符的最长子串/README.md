# 题后总结

## 知识点
滑动窗口 + 哈希表

## 思路
1. 使用双指针维护一个滑动窗口，左指针 l 和右指针 r
2. 用哈希表记录每个字符最后出现的位置
3. 当遇到重复字符时，移动左指针到重复字符上一次出现位置的下一位
4. 每次移动后更新最大长度

## 关键点
1. 哈希表存储字符的索引位置，便于快速定位重复字符
2. 遇到重复字符时，需要清除窗口左侧被移出的字符记录
3. 窗口大小 = r - l + 1
4. 时间复杂度 O(n)，空间复杂度 O(min(m,n))，m 为字符集大小

## 易错点
1. 空字符串需要特殊处理，返回 0
2. 删除哈希表记录时要循环删除，确保窗口内无重复
3. 最大值的更新时机：在确定窗口无重复后更新
